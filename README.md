# Spring Guides

## 资源结构

Spring 官网 Guides 系列教程，按照 Spring in Action (5th Edition) 的目录结构，资源分5个部分：

* Part 1: Foundational Spring
* Part 2: Integrated Spring
* Part 3: Reactive Spring
* Part 4: Cloud-Native Spring
* Part 5: Deployed Spring
